Прод: **https://birliy.uz** · хостинг Railway · деплой командой `railway up`.

## 1. SSL-сертификат для birliy.uz ✅
- Добавили DNS-запись CNAME от Sectigo (через Cloudflare) для проверки домена.
- **Сертификат выпущен, сайт работает по https.**

## 2. Telegram-бот заявок — фото одним альбомом ✅
- Раньше бот слал патент/паспорт/магазин **тремя отдельными сообщениями**.
- Теперь — **одним альбомом** с подписью (бизнес, владелец, телефон, город) на первом фото.
- Кнопки статуса (В работу / Демо / Выигран / Проигран) остались отдельной карточкой.

## 3. Загрузка документов прямо на сайте ✅
Форма заявки на сайте теперь принимает **3 фото**: патент, паспорт директора, витрина магазина.
- При отправке фото уходят **в группу «BirLiy Leads»** одним альбомом.
- Приватная копия документов сохраняется в **Supabase** (хранилище `lead-docs`, не публичное — паспорта под защитой).
- Заявка пишется в базу. «Голых» заявок без документов больше нет.
- Проверено вживую: фото дошли и в группу, и в Supabase.

**Supabase (новый сервис):** проект `birliy`, регион Мумбаи, бакет `lead-docs` (приватный).
Документы клиентов смотреть тут: Supabase → Storage → lead-docs → leads/{номер заявки}/.

## 4. Доступность и мобильная версия ✅
- Добавил видимую рамку фокуса для навигации с клавиатуры.
- Поправил FAQ и поля формы для скринридеров.
- Увеличил мелкие кнопки до удобного размера на телефоне.
- Сделал строку про защиту данных читаемой (был слабый контраст).
- Проверил всю страницу на телефоне (390px) — **горизонтальной прокрутки нет, всё аккуратно**.

## 5. SEO под узбекский рынок ✅
**Главное:** раньше для Google узбекской версии сайта **не существовало** (язык переключался кнопкой через JavaScript, Google видел только русский).
- Теперь **адрес = язык**: `birliy.uz` — узбекский (основной), `birliy.uz/ru` — русский.
- Связал версии тегами hreflang, добавил обе в карту сайта.
- Узбекские ключевые слова (kassa dasturi, ombor dasturi, do'kon uchun kassa и т.д.).
- **Google Search Console:** домен подтверждён (через Cloudflare), карта сайта отправлена, главная страница уже в индексе Google. Русская — в очереди на индексацию.

## 6. Правки по тексту ✅

- **Полностью убрал старый партнёрский бренд** со всего сайта (футер, блок доверия, SEO, структурные данные). Заменил на нейтральные доводы (безопасность по PIN, журнал действий, «сделано для Узбекистана»).

## 7. Концепт-страница (живой прототип) — полировка ✅
Страница **birliy.uz/concept** — премиум-лендинг с кликабельным макетом приложения (не скриншот, всё нажимается).
- Кнопка **«Смотреть продукт»** теперь ведёт прямо на живое демо.
- Исправил ошибку: **«QR-сканер» → «Сканер штрих-кодов»** (сканер читает штрих-коды, не QR).
- **Заполнил прототип:** 10 товаров, у каждого картинка-эмодзи (💧☕🍞🧃🍫🍬🍟🚬⚡🔥) вместо пустых квадратов.
- **Переписал блок «боли»** под посыл менеджера: *«Сколько товара осталось? Сколько заработали? С тетрадью точного ответа нет»* → BirLiy всё чинит + **мы подключаем за день, заносим товары вместе с вами, учим кассира за полчаса**.
- Убрал «роботные» длинные тире из верхней части (герой, боль, демо).
