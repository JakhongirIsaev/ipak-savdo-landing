# Seedance (CapCut / Dreamina 2.0) — промпты для BirLiy

> Создано 2026-06-05. Под **Dreamina Seedance 2.0** в CapCut. Важное ограничение модели: **лица людей не поддерживаются** → все ролики **без лиц**: руки, телефон, полки, сцена. Это совпадает с бренд-стилем (спокойно, по делу) и убирает риск «AI-лица».
>
> **Как пользоваться:** вставляешь английский промпт в поле Dreamina → генерируешь 1 тест на дешёвой настройке → смотришь → потом батч. Текст/логотип/CTA **накладываешь поверх в CapCut** (внутри AI текст не пишем — кириллицу он рендерит плохо). У тебя ~230 бесплатных кредитов; видео дороже картинок — не жги вслепую.

## Бренд-рамки (держать в каждом ролике)
- Формат **9:16 вертикаль**, длина **7–8 сек**, движение спокойное (медленный наезд, лёгкий параллакс).
- Свет тёплый, естественный. Тона бумажные (#F6F7F4 настроение). Один зелёный акцент #03B73D — только в оверлее, не в AI-сцене.
- Локация — Ташкент/Центральная Азия: дукан, минимаркет, аптека, чайхана.
- **Нельзя:** лица крупным планом, фиолетовые градиенты, «космос», свечения, глянец, сток-улыбки, читаемый текст в кадре, перерисованный интерфейс приложения (экран держим размытым / не в фокусе).
- Звук убираем — Telegram часто смотрят без звука; смысл несёт оверлей.

## Чего НЕ делать в Seedance
- **Экран приложения крупно** — AI перерисует интерфейс криво. Для демо продукта («продажа за 15 сек», отчёты) используй **реальную запись экрана**, Seedance только на интро/аутро.
- **Лица** — модель их не поддерживает; держим руки/со спины/детали.
- **Текст в кадре** — кириллицу AI портит; весь текст накладываем в CapCut.

Seedance (чистая сцена, без текста) → CapCut: оверлей RU/UZ + зелёный акцент #03B73D + лого-аутро + CTA → экспорт 9:16 → пост по календарю.
