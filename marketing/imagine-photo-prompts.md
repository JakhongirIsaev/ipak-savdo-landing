# BirLiy — промпты для реалистичных фото (Imagine 2.0)

4 фото для лендинга/концепта. Генерируй каждое по своему промпту, потом пришли мне файлы — я вставлю на сайт.

**Общие требования (важно для всех 4):**
- Соотношение сторон **4:3 горизонтально** (landscape), не меньше ~1600px по ширине.
- **Узбекистан, Ташкент**, аутентичные местные люди (узбеки), без «стоковой» постановки.
- Тёплый дневной свет, спокойно и по-настоящему, лёгкая киношность.
- **Без текста, без логотипов, без брендов** на экранах и упаковках.
- Фото реалистичное (photoreal), не иллюстрация, не 3D-рендер.
- Negative (если поле есть): `text, watermark, logo, brand, distorted hands, extra fingers, cartoon, 3d render, plastic skin`
